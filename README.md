# music-genre-classification
Deep learning based music genre classification using CNN in Python

## Overview
This project classifies music samples into 10 distinct genres (Rock, Jazz, Pop, Hip-hop, Blues, Classical, Reggae, Country, Metal, Disco) using a Convolutional Neural Network (CNN). 
The model learns patterns from audio frequency features.

## Dataset Source
The GTZAN Music Genre Dataset was obtained from Kaggle (URL: https://www.kaggle.com/datasets/carlthome/gtzan-genre-collection ).
